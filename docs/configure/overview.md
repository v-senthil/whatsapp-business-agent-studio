# Configure overview

The phone dashboard is the hub for one phone number. This page explains how the layout is structured and how to move between sections.

## What you'll do

- Learn the shape of the dashboard.
- Understand the Configure, Ship, and Advanced groups in the sidebar.
- Find the buttons for exporting and importing the whole agent config.

## Layout at a glance

- A **header** at the top holds the theme toggle, the account menu, and the read-only toggle.
- A **breadcrumbs** row below the header shows Business, WABA, and phone context.
- A **sidebar** on the left lists every section, grouped into **Configure**, **Ship**, and **Advanced**.
- The main area on the right shows the current section.

## Sidebar groups

**Configure**

- Overview (the current page), with the [onboarding checklist](../getting-started/onboard-your-first-agent.md).
- [Settings](settings.md), rollout, handoff, follow-up, and AI audience.
- [Business info](business-info.md), description, email, and policies.
- [Skills](skills.md), the agent's scenario instructions.
- Knowledge, tabs for [files](knowledge-files.md), [websites](knowledge-websites.md), [FAQs](knowledge-faqs.md), and [allowlist](knowledge-allowlist.md).
- [Connectors](connectors.md), external APIs the agent can call.
- [Generate from doc](generate-from-document.md), AI-assisted drafting from a document.

**Ship**

- [Test](../ship/test-chat.md), chat with the agent.
- [Scenarios](../ship/scenarios.md), save and replay conversations.
- [Evals](../ship/evals.md), run Meta eval cases.
- [Agent events](../ship/agent-events.md), trigger partner-defined events.
- [Thread control](../ship/thread-control.md), release or pass control to a human.
- [Webhooks](../ship/webhooks.md), receive Meta events on a live page.

**Advanced**

- [Playground](../ship/playground.md), a free-form request builder.
- [API docs](../advanced/api-reference.md), browse the API reference inside the app.
- AI provider settings, at the top-level Settings page. See [AI provider settings](../advanced/ai-provider.md).

## Overview page contents

The overview shows, from top to bottom:

- The onboarding checklist, when the agent is not fully set up.
- A summary of the phone's eligibility status.
- A list of configured agents on this phone.
- Two header buttons: **Export config** and **Import config**. See [Config export and import](../advanced/config-export-import.md).
